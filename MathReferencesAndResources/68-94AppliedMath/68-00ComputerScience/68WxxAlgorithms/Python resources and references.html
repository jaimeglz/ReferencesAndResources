Python resources and references

This map presents the organization of the Python references and resources. In the web page, the actual hyperlinks are either above or below the images of their respective nodes; whereas in the XMind representation the icons are either hyperlinks or notes.

The hyperlink in this node (or at the top of the web page) points to the official web site of the Python Software Foundation.

The codes "68N15" and "68W30" correspond to the MSC2010 classification for:

68 Computer science, and under this topic 68N is for software, 68W for algorithms.

68N15 corresponds to programming languages.

68W30 corresponds to symbolic computation and algebraic computation.
